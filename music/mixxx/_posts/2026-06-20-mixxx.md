# 🎧 Mixxx - The Open Source DJ Studio

## 🔑 Introduction

Not every music creator wants to compose songs from scratch.

Some want to mix, blend, remix, and perform music live.

That is where DJ software comes into the picture.

For Linux users, **Mixxx** has become one of the most respected open-source DJ applications available.

For many years, it has served as the Linux alternative to commercial DJ software such as **VirtualDJ** on Windows.

---

# 🎵 What Is Mixxx?

Mixxx is a free and open-source DJ application designed for live music mixing and performance.

It allows DJs to:

- Mix multiple tracks
- Beat match songs
- Create playlists
- Use DJ controllers
- Apply audio effects
- Broadcast live streams
- Record DJ sets

Mixxx brings professional DJ functionality to Linux without requiring expensive software licenses.

---

# 🖥️ Mixxx And Linux

Mixxx has long been a favourite among Linux users because it:

🔑 Is free

🔑 Supports many DJ controllers

🔑 Runs on multiple Linux distributions

🔑 Has a strong community

🔑 Continues to improve

Whether you use MX Linux, Debian, Ubuntu, Fedora, or another distribution, Mixxx provides a capable DJ environment.

---

# 🪟 VirtualDJ On Windows

For many Windows users, **VirtualDJ** became one of the most popular DJ applications.

VirtualDJ offers:

- Advanced mixing tools
- Video mixing
- Karaoke support
- Controller integration
- Professional performance features

For years, VirtualDJ helped introduce thousands of people to digital DJing.
