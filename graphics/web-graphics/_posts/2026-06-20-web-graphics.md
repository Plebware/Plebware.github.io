# 🌐 Web Graphics - Designing For The Digital World

## 🔑 Introduction

The internet is a visual environment.

Before people read a paragraph, they often notice:

- Images
- Icons
- Colours
- Layout
- Visual style

Web graphics help transform information into an experience.

A good graphic does not only decorate a website.

It communicates.

---

# 🎨 What Are Web Graphics?

Web graphics are visual elements created for online use.

They include:

- Website banners
- Icons
- Buttons
- Background images
- Illustrations
- Logos
- Infographics
- Social media images
- Thumbnails

They are designed specifically for screens.

---

# 🖥️ The Difference Between Print And Web Design

Print and web graphics share many principles, but they work differently.

Print:

- Fixed size
- Physical material
- High resolution requirements

Web:

- Different screen sizes
- Different devices
- Faster loading requirements
- Interactive environments

A web graphic must be flexible.

---

# 🔑 The Foundations Of Good Web Graphics

## 1. Clarity

A visitor should understand the message quickly.

Ask:

"Can someone understand this in a few seconds?"

Good web graphics reduce confusion.

---

## 2. Consistency

A website should have a visual language.

Keep consistent:

- Colours
- Fonts
- Shapes
- Style
- Image choices

Consistency creates identity.

---

## 3. Performance

Beautiful graphics must also load quickly.

Large images can slow down websites.

Good practices:

- Optimise file sizes
- Use suitable formats
- Resize images correctly

---

# 📐 Common Web Graphic Types

## Hero Images

Large images near the top of a page.

Purpose:

- Create impact
- Introduce a topic
- Establish mood

---

## Icons

Small symbols that communicate actions.

Examples:

- Search
- Menu
- Settings
- Navigation

Good icons are simple and recognisable.

---

## Banners

Used for:

- Announcements
- Events
- Promotions
- Important messages

---

## Thumbnails

Small previews.

Important for:

- Articles
- Videos
- Books
- Projects

A strong thumbnail attracts attention.

---

# 🖌️ Web Graphic Design Principles

## Balance

Elements should feel organised.

Avoid overcrowding.

---

## Contrast

Important elements should stand out.

Contrast can come from:

- Size
- Shape
- Position
- Colour

---

## Space

Empty space is useful.

It gives the design room to breathe.

---

# 🛠️ Creating Web Graphics

A simple workflow:

## Step 1: Define The Purpose

Ask:

What is this graphic supposed to do?

---

## Step 2: Choose The Format

Common formats:

PNG:
- Transparency
- Logos
- Icons

JPEG:
- Photos
- Complex images

SVG:
- Scalable graphics

WebP:
- Modern web images

---

## Step 3: Design

Create the visual.

Focus on:

- Message
- Readability
- Style

---

## Step 4: Test

Check:

- Desktop
- Tablet
- Mobile

A graphic should work everywhere.
