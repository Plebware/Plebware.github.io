# 🔑 What Is a Web Browser?

A web browser is a software application used to access and interact with the internet.

A browser allows you to:

* Visit websites
* Search for information
* Read articles
* Watch videos
* Use online tools
* Download files
* Communicate online

The browser is the bridge between you and the World Wide Web.

---

# 🔑 How Does a Browser Work?

When you enter a website address, the browser:

1. Connects to the website server
2. Requests the webpage
3. Receives the information
4. Displays the page on your screen

The browser understands web technologies such as:

* HTML
* CSS
* JavaScript

and turns code into the websites people see.

---

# 🔑 The Main Parts of a Browser

## Address Bar

The address bar allows you to:

* Enter website addresses
* Search the internet
* Open pages

---

## Tabs

Tabs allow multiple websites to be open at the same time.

You can:

* Switch between pages
* Organize research
* Compare information

---

## Bookmarks

Bookmarks save useful websites for later.

They are helpful for:

* Research
* Favourite websites
* Important resources

---

## Extensions

Extensions add extra features.

Examples:

* Password managers
* Ad blockers
* Translation tools
* Productivity tools

---

# 🔑 Popular Web Browsers

## Mozilla Firefox

Firefox is an open-source browser known for focusing on:

* Privacy
* User control
* Customization

It is popular among users who prefer open technology and greater control over their browser.

---

## Brave

Brave is a browser designed with privacy features built in.

Features include:

* Tracker blocking
* Faster page loading
* Privacy-focused settings

Brave is based on Chromium technology.

---

## Google Chrome

Chrome is one of the most widely used browsers.

It is known for:

* Speed
* Large extension library
* Integration with Google services

Chrome uses the Chromium browser engine.

---

## Microsoft Edge

Edge is Microsoft's modern browser.

It provides:

* Chromium compatibility
* Windows integration
* Productivity features

It replaced the older Internet Explorer browser.

---

## Vivaldi

Vivaldi is designed for users who want deep customization.

Features include:

* Advanced tab management
* Custom layouts
* Built-in tools
* User interface controls

It is popular among power users.

---

# 🔑 Browser Engines

A browser engine is the technology that displays websites.

Common engines include:

## Chromium

Used by:

* Chrome
* Edge
* Brave
* Vivaldi

Chromium provides compatibility with many modern websites.

---

## Gecko

Used by:

* Firefox

Gecko is Mozilla's independent browser engine.
